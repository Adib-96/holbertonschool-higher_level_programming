=======================
how to use 1-my_list.py
=======================

This module define a class ``MyList`` that inherits functionality from ``list``


Instantiation
=============

::

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> type(my_list)
<class '1-my_list.MyList'>

::

>>> print(my_list)
[]

Or a single arg, must be an iterable

::

>>> my_list = MyList([10, 20, 30])
>>> print(my_list)
[10, 20, 30]

::

>>> my_list = MyList(None)
    Traceback (most recent call last):
    TypeError: 'NoneType' object is not iterable


A ``TypeError`` will be raised for any more than one instantiation arguments.

::

    >>> my_list = MyList([1, 2], [3, 4])
    Traceback (most recent call last):
    TypeError: list() takes at most 1 argument (2 given)


UseCase
=======

inheritance is present here , MyList inherit from list.
Append methode will be iherited from list class to add
elements in a list.

::

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(3)
>>> print(my_list)
[1, 2, 3]
